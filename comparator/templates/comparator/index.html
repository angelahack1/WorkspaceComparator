⚙ Engine Settings

LLM failure limit 3
Consecutive LLM failures (timeouts, unreachable server, garbage answers) before AI arbitration is switched off for the rest of the run.
Text charset by side AUTO
Auto detects each file independently. Override one side only when a legacy charset is ambiguous. Binary content remains binary regardless of this setting.
Max LLM candidates per file 3
Only the top-N most similar candidates for each left file are sent to the LLM. 0 disables AI arbitration entirely.
LLM similarity noise floor 15%
Candidate pairs whose deterministic similarity is below this never reach the LLM — they are considered noise.
Renamed-file content threshold 60%
Files with completely different names are still matched when their content similarity reaches this percentage.

🚫 Exclusions

Matching files and folders are not compared. Keep Show excluded checked to retain them in the Ignored Files section, or uncheck it to hide them from the table. Wildcards: * (any text) and ? (one character). Plain patterns match the name anywhere in the tree (e.g. *.min.js, generated); patterns with / match the relative path (e.g. docs/internal, src/*/legacy). Use the switch to disable a pattern without deleting it.
