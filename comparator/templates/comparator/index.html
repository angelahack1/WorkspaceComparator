⚙ Engine Settings

LLM failure limit 3
Consecutive LLM failures (timeouts, unreachable server, garbage answers) before AI arbitration is switched off for the rest of the run.
Max LLM candidates per file 3
Only the top-N most similar candidates for each left file are sent to the LLM. 0 disables AI arbitration entirely.
LLM similarity noise floor 15%
Candidate pairs whose deterministic similarity is below this never reach the LLM — they are considered noise.
Renamed-file content threshold 60%
Files with completely different names are still matched when their content similarity reaches this percentage.

🚫 Exclusions

Matching files and folders are removed from the comparison entirely — they are never scanned, matched, or shown. Wildcards: * (any text) and ? (one character). Plain patterns match the name anywhere in the tree (e.g. *.min.js, generated); patterns with / match the relative path (e.g. docs/internal, src/*/legacy). Use the switch to disable a pattern without deleting it.
